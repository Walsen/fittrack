# Summit CrossFit Box

## Location
789 Industrial Blvd, Warehouse District

## Hours
Monday–Friday: 5:30 AM – 8:30 PM
Saturday: 7:00 AM – 1:00 PM
Sunday: Closed

## Overview
Summit CrossFit Box is a dedicated CrossFit affiliate gym housed in a converted 8,000 sq ft warehouse. The open floor plan is optimized for high-intensity functional training, Olympic lifting, and metabolic conditioning. All workouts are coach-led in group class format.
