# Zenith Wellness Studio

## Location
321 Elm Street, Riverside

## Hours
Monday–Friday: 6:00 AM – 10:00 PM

## Overview
Zenith Wellness Studio is a boutique fitness facility focused on low-impact training, mobility, and recovery. The 6,000 sq ft space features a Pilates studio, yoga room, and a recovery lounge. Ideal for those prioritizing flexibility, rehabilitation, and mind-body fitness.
